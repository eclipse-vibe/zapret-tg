# zapret-tg

Утилита для локального обхода DPI-блокировок Telegram на территории РФ. Восстанавливает стабильный доступ, звонки и быструю загрузку медиа без покупки сторонних VPN или прокси-серверов.

## ✨ Особенности / Фичи
* **Максимальная скорость**: Скорость скачивания картинок и видео не режется и остается такой, какая у вас скорость интернета.
* **Локальный DPI-bypass**: Утилита модифицирует пакеты прямо на вашем ПК, заставляя ТСПУ пропускать трафик Telegram.
* **Низкий пинг**: Прямое подключение сохраняет минимальный отклик в звонках и чатах.

## ⚠️ Известные ограничения (В разработке)
На текущий момент проект находится в стадии активного тестирования, поэтому:
1. **Не работают Telegram Mini Apps** (встроенные приложения и игры внутри чатов).
2. **Не открываются ссылки вида `t.me`** в браузере.
*Исправление этих нюансов планируется в следующих обновлениях.*

## 🚀 Инструкция по запуску (В один клик)

Для работы утилиты пользователю больше ничего настраивать не нужно. Просто следуйте двум шагам:

1. Скачайте архив с проектом и распакуйте его.
2. Запустите файл **`ZAPRET_TG.bat`** (желательно от имени Администратора).
3. **Запустите Telegram** и пользуйтесь. 

*Больше ничего трогать и настраивать не нужно! Окно батника должно оставаться открытым, пока вы пользуетесь Telegram.*
